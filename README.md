# Octree-Test
A C89 OpenGL 1.1 demo of octree map compiling and rendering

---

Build and run with
```
make -j$(nproc) run
```

---

```
CONTROLS:
    Esc    - Exit
    W      - Move forwards
    A      - Strafe left
    S      - Move backwards
    D      - Strafe right
    Space  - Move upwards
    LShift - Move downwards
    LCtrl  - Move faster
    M      - Toggle mouse grab
    R      - Reload map
    1      - Render normal
    2      - Render overdraw heatmap
    3      - Render overdraw heatmap with depth test disabled
```
